Fragment​

每一個 React 組件都只能回傳一個元素，當我們希望回傳的內容有多個元素的時候，可以透過Fragment把多個元素 Group 再一起，但是不會增加額外的 div。

function Post() {
  return (
    <>
      <PostTitle />
      <PostBody />
    </>
  );
}
